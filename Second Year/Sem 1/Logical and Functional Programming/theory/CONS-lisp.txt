În Lisp, funcția CONS este una dintre funcțiile fundamentale pentru manipularea listelor și a structurilor de date. Numele CONS provine de la "construct", și este folosită pentru a crea perechi (denumite și cons cells sau dotted pairs).

Funcționalitatea CONS
CONS ia două argumente și le combină într-o pereche. Primul argument devine partea car a perechii, iar al doilea argument devine partea cdr.

Sintaxă
lisp
Copy code
(CONS element1 element2)
element1: Acesta este primul element sau partea car a perechii.
element2: Acesta este al doilea element sau partea cdr a perechii.
Exemple de Utilizare
Crearea unei Perechi Simple:

lisp
Copy code
(CONS 'a 'b)
;; Rezultat: (A . B)
Aceasta creează o pereche unde car este A și cdr este B.

Construirea unei Liste:
CONS poate fi folosită pentru a construi liste prin adăugarea unui element la începutul unei liste existente.

lisp
Copy code
(CONS 'a '(b c d))
;; Rezultat: (A B C D)
În acest caz, a este adăugat la începutul listei (b c d).

Crearea de Structuri de Date Mai Complexe:
Poți utiliza CONS pentru a construi structuri de date complexe, cum ar fi arbori sau liste îmbricate.

lisp
Copy code
(CONS 'a (CONS 'b 'c))
;; Rezultat: (A . (B . C))
Caracteristici Importante
CONS este esențială pentru construcția listelor în Lisp, deoarece orice listă este de fapt o serie de perechi create cu CONS.
În mod tradițional, în Lisp, o listă este o serie de perechi conectate unde cdr fiecărei perechi (cu excepția ultimei) este legată de următoarea pereche din listă, iar cdr ultimei perechi este de obicei NIL (reprezentând sfârșitul listei).
Prin urmare, CONS este o funcție de bază în Lisp, utilizată atât pentru construirea listelor, cât și pentru reprezentarea altor structuri de date mai complexe.
